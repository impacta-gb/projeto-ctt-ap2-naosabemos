# Projeto de Documentação - Linguagem Go 🚀

Este projeto consiste em um site de documentação completo sobre a linguagem de programação Go (Golang), cobrindo desde a instalação básica até conceitos avançados como concorrência (Goroutines e Channels) e testes automatizados.

O site foi construído utilizando a ferramenta **Zensical** para geração de páginas estáticas a partir de arquivos Markdown.

## ⚙ Automações CI/CD

Nosso workflow automatiza o teste, a construção e a publicação do site de documentação. Ele usa cache e matriz de testes para garantir velocidade e estabilidade, além de regras de segurança que impedem o deploy acidental durante a fase de revisão de código (Pull Requests).
